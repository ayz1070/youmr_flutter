## 📱 프로젝트 소개

YouMR(You Music Room)은 기타 동호회 '여민락'을 위한 통합 관리 앱입니다. 출석 체크, AI 기반 음악 학습 지원, 커뮤니티 기능을 통해 동호회원들의 음악 활동을 더욱 풍부하게 만들어줍니다.

## ✨ 주요 기능

### 🎯 출석 체크 시스템
- 출석 체크
- 출석 통계 및 이력 관리
- 개인별 출석 현황 확인

### 🤖 AI 음악 도우미 (Gemini)
- 코드 진행 분석 및 추천
- 악기 연주 팁 제공
- 음악 이론 학습 지원
- 실시간 음악 관련 질의응답

### 👥 커뮤니티
- 동호회원 간 소통
- 공지사항 및 일정 관리
- 연주 영상 공유
- 음악 관련 정보 교류

## 🛠 기술 스택

- **프론트엔드**: Flutter
- **백엔드**: Firebase
- **인증**: Firebase Auth, Kakao Login
- **AI**: Google Gemini
- **상태관리**: Riverpod
- **라우팅**: Go Router
- **푸시알림**: Firebase Cloud Messaging

## 🚀 시작하기

### 필수 조건
- Flutter SDK 3.7.0 이상
- Firebase 프로젝트 설정
- Gemini API 키

### 설치 방법
```bash
# 저장소 클론
git clone https://github.com/your-username/youmr_flutter.git

# 의존성 설치
flutter pub get

# 환경 변수 설정
cp .env.example .env
# .env 파일에 필요한 API 키들을 설정해주세요

# 앱 실행
flutter run
```
